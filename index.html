AutoEasy Rider

Project Breif:

This app is developed for Auto Riders who are searching for auto nearby.
User's are requested to complete a simple Registration process
******************************************************************
1. Users need to Register by pressing on Register now.
2. After that user just need to login with the credentials.
3. Location permission is requested in order to find nearby drivers.
Features
******************
-> Users can find nearby drivers by clicking on "Search for Drivers" in bottom right menu in Map page
-> Google Maps included, Users can see their current location
-> If there are more than 3 request from the users in one area, All the Auto Drivers in that area will be notified instantly
-> Emergency Call included incase of any Emergency.
-> List of nearby Drivers will be displayed to the users, from there users can contact to Drivers just with one-click
The UI has both Light mode and Dark mode

Project Info

  • Date - 2020
  • Tools - Flutter,dart,firebase
  • Web - visit the playstore
